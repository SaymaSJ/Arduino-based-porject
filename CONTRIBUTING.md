# Contributing to Arduino Projects

**How to add your projects to this repository:**

## Steps

1. **Create a project folder** in the appropriate category
2. **Add your code files**:
   - main.ino (Arduino code)
   - config.h (configuration)
   - circuit_diagram.fzz (Fritzing schematic)

3. **Document your project**:
   - README.md with description
   - How to replicate
   - Components used
   - Results/photos

4. **Commit with clear messages**:
   ```
   git add .
   git commit -m "Add: Temperature sensor project"
   git push
   ```

5. **Add to projects list** in main README.md

---

## Project Documentation Template

**Use this template for each project:**

```markdown
# Project Name

## Description
(What does this project do?)

## Hardware Used
- Component 1 (Model)
- Component 2 (Model)

## How It Works
(Your explanation)

## Setup Instructions
(Step-by-step setup)

## Results
(Photos, data, outcomes)

## What I Learned
(Key learnings)

## Future Improvements
(What could be better)
```
